# 書本資訊

# 書名：我在這裡，山在那邊

# 作者：呂忠翰 (阿果)

# 出版：城邦文化

# ISBN：978-986-408-560-6

# 簡介

本書主要是在敘述作者在爬山中發生的故事，用簡單明瞭的口吻來闡述自身經歷
從全人中學到最後的 K2 世界第二高峰
從台灣到國外挑戰 8000 公尺的高山
從一營到登頂的經歷
讓自己增加了多少歷練、幫自己與身邊的朋友成長
認知到自己的極限，不越界，不讓自己受傷
看到朋友的困難，捨身相救
以作者說故事及與友人對話的型態
經過了作者純樸及善良的文筆 寫下這本經典

# 印象深刻

# 第三章 - 生與死的契約

這篇在敘述作者第一次攀爬位於 巴基斯坦喀拉崑崙山脈 (8000m) 世界第十三高峰

無氧攀登者在 8000m 的山脈，為了避免高山症，通常會先設置 4~5 個營地
讓登山者來回走動以適應高度，然而
在 7000m 以上的高度，人體會因為缺氧而無法自然回復
所以通常會需要在短時間內迅速登頂
很多人問作者為何選擇「無氧攀登」，但作者本人並非是想要挑戰甚麼
反而是想尋找一種野性的魅力，用自己最完整的身體來感受大自然

作者在經歷千辛萬苦後終於登頂，但接下來的事才真正令我印象深刻
因為一個隊友 (以下稱為文辰) 的體力不支而直接從山坡上滑落幾十公尺
直到中國境內才停下來，雖然作者最終有找到他，但
通常在山中迷失且無法行動是很難獲得救援的
作者也思考了很久，最後仍然下定決心陪在文辰身邊
最終，文辰也因此獲救

看完這段故事，我也在想...

# 如果是我呢？我也會留下嗎？

但運氣沒那麼好呢？一起凍死在山上？
還是陪著摯友走完最後一程？
我有辦法像作者一樣冷靜處理嗎？

到最後我有個結論
不論是誰，我會不惜生命的留在他身邊，
即使他的生命走到盡頭，我也會陪伴他到最後
孤獨的離去實在是太哀傷了

英國有位知名脫口秀演員在上台表演時突然暴斃身亡
台下的觀眾都還以為他在開玩笑
直到 30 秒後，大家才發現事情不對勁
我想，這應該是那位演員最精采的演出了吧！

在做著自己喜歡的事時離去，這感覺比病死好多了吧！
我會說服自己這樣想...

# 第六章 - 最後的三百米

這篇在敘述作者攀爬位於 道拉吉里峰 (8000m) 世界第七高峰

道拉吉里又被稱作為雪白的山，很容易受天氣影響
而作者為了此次登頂已經花了將近十日在等壞天氣過去
所以這次就算是接到警報，仍然要前往登頂
雖然途中帳棚還被雪崩壓跨，整隊都士氣低迷

在登頂過程中，隊員一個一個開始下撤
直到最後剩下四個成員，雖然大家都想登頂
但由於風實在是不太賞臉，雪甚至深到可以埋住大腿
隊友也快撐不下去了
即使眼睛已經可以看到頂端
即使只剩下一個斜坡
但最終還是安全因素只好返回

作者認為團隊不要出事才是真正的快樂
所以就算自己可以登頂，但
團隊沒有一起登頂那又算什麼
登頂後，還有回程要走...
身為團隊領導者，有太多事情要考慮了
理性的思考下一步，即使是在困境中

能夠為了團隊犧牲登頂的機會
作者可以付出一切扛下責任，很值得令人敬佩
對作者來說，登山最重要的其實是過程
如何處理在山中的困境，
如何協助隊友，團隊合作的樂趣
山中的風景等等
都比登頂來的值得
